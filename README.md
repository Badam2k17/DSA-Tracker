# DSA Tracker 🚀

A simple and clean web app to track solved DSA problems.

## 🔥 Features

- Add problems
- Mark as solved ✅
- Delete problems ❌
- Progress tracking 📊
- Data saved using localStorage

## 🛠 Tech Stack

- React
- JavaScript
